# CityMind — Urban Policy Simulation Platform

## One Line
Governments simulate infrastructure policies on a digital twin of Lahore 
before spending billions in real life.

## The Core Flow
1. User opens app — sees live dark map of Lahore
2. User picks a policy (flyover, hospital, drainage)
3. User draws location on map, sets budget + parameters
4. Clicks Simulate
5. Map animates — zones light up red/green showing impact
6. AI generates recommendation with verdict + alternatives
